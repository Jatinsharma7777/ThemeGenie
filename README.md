# ThemeGenie

ThemeGenie is a browser extension that allows users to completely customize the appearance of virtually any website through intelligent theme generation, live editing, and custom styling. Whether you want a minimal interface, a glassmorphism design, or a complete visual overhaul, ThemeGenie provides the tools to transform websites without manually writing complex CSS.

The extension supports live theme previews, custom CSS editing, theme importing, and website-specific customization, making it suitable for both casual users and advanced theme creators.

## Features

* Generate custom themes using natural language prompts.
* Customize the appearance of almost any website.
* Live preview before applying changes.
* Built-in CSS editor for manual customization.
* Import and export custom themes.
* Website-specific theme management.
* Compatible with custom CSS themes such as DarkMatter for Discord.
* Modular architecture for future expansion.

## Installation

1. Download or clone this repository.
2. Extract the project if downloaded as a ZIP.
3. Open **Google Chrome** and navigate to `chrome://extensions/`.
4. Enable **Developer Mode** using the toggle in the top-right corner.
5. Click **Load unpacked**.
6. Select the **ThemeGenie** project folder.
7. The extension will be installed and ready to use.

## Example

For Discord, you can import custom themes such as **DarkMatter** or create your own theme using ThemeGenie's editor and customization tools.
